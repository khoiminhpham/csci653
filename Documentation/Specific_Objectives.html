Step-by-step to the Goal: 

1. First, we'll need to set up containerization environment on a super computer.
2. Choose a program we've seen in this class, such as image processing, or energies simulation, then add time execution measurements. 
3. Run it on HPC environment alone, and scale up the amount of recourses, until seeing no more improvement in running time.
4. Install required library to run this program in the new container (probably mpich, and openMP). Run the program on this container as describe in step (3), keep all the parameters unchanged.
5. Now we can compare the execution time between 2 test cases, the trade-off percentage is expected to be lower than 10%.
