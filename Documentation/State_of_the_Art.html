Current State of the Art.

Containerization on HPC environment is still relatively new and not fully supported.
A few Containerization tool that works in HPC: Singularity, Shifter, CharlieCloud etc.
Since Docker image can be converted to Singularity, I'll choose Singularity.
Portability maybe limited, therefore, it'd be nice if we can find a (parallel computing) program that can run on both environment.

Some other approaches include 
     Deploy HPC services inside Docker Container
     Use DockerSwarm to replace HPC workload manager
     etc.
